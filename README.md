# LLM Portfolio Examples

Two interactive demo projects built with Next.js and TypeScript.

## Insurance Dashboard

A fixed income portfolio risk dashboard featuring:
- 50 USD bond portfolio (30 Government, 20 Corporate)
- Duration, convexity, PV01/DV01/CR01 analytics
- Monte Carlo simulation (Vasicek IR + Gaussian Copula credit model, 1,000 scenarios)
- Portfolio rebalancing controls
- Supabase persistence
- Python bond metrics calculator

## How an LLM Works

An interactive demo showing how a large language model predicts the next word:
- Real-time parameter exploration (temperature, top-p, frequency/presence penalty, etc.)
- Token probability distribution visualization
- RAG context injection demo
- System prompt comparison
- Pre-cached API responses (~300+ parameter combinations)
- No runtime API calls required
